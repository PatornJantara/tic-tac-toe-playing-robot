# Tic-Tac-Toe-Playing-Robot
## An automatic tic tac toe playing robotic arm

Requirements

- Software
 - Python packages
  - numpy
  - cv2
  - serial
 - Arduino libraries
  - Servo.h
  
- Hardware
 - Arduino Mega x 2
 - BTS 7960
 - Webcam x1
 - Powersupply 5v / 24 v

- Robotic arm configuration
5 DOF Robotic arm ( joint 1,2 are drive by DC motor and Others are driven by Servo motor)
